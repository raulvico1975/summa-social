# Client

Client Next.js minim per a la Fase 1 del `web-agent` text-first.

## Scope actual

- UI de conversa en text
- render de `rich UI` segons contracte JSON
- consum directe del backend Python del pilot

## Variables

```bash
cp .env.example .env.local
```

`NEXT_PUBLIC_WEB_AGENT_API_URL` apunta per defecte a `http://127.0.0.1:8787`.

## Run local

```bash
npm install
npm run dev
```
