# Phase Review Template

Utilitza aquesta plantilla al tancament de cada fase.

## 1. Problema

- Quin problema concret de Summa hem atacat?

## 2. Hipotesi de valor

- Per que el canal triat havia de ser millor que text, veu, FAQ o UI classica?

## 3. Scope executat

- Que ha entrat realment?
- Que hem deixat fora?

## 4. Evidencia minima

- Quina prova hem obtingut?
- Que ha notat una persona usuaria, comercial o operativa?

## 5. Risc principal

- Quin risc queda viu?
- Com el controlem abans de la fase seguent?

## 6. Decisio del product-director-agent

- `GO`
- `GO WITH TRIM`
- `HOLD`
- `NO-GO`

## 7. Seguent pas

- Quin es el scope maxim acceptable de la fase seguent?
