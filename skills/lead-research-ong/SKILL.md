# Lead Research ONG

## Quan usar-la

Usa aquesta skill quan Raul doni:

- nom d'una entitat
- web
- Excel o CRM amb candidats
- resolucio publica de subvencions
- llista d'entitats a prioritzar

## Fonts prioritaries

Consulta o estructura primer fonts oficials i recents:

- web oficial de l'entitat
- memoria anual
- transparencia
- projectes actius
- subvencions publiques
- equip o junta
- contacte operatiu publicat
- canals sectorials o notes publiques

Si no hi ha font suficient, indica que no s'ha pogut verificar. No omplis buits amb suposicions.

## Procediment

1. Identifica web oficial i domini.
2. Busca memoria, transparencia o comptes si existeixen.
3. Detecta sector i activitat recent.
4. Busca indicis de mida: pressupost, subvencions, nombre de projectes o equip.
5. Identifica persones responsables si son publiques.
6. Separa senyals d'encaix de riscos de descart.
7. Formula una pregunta d'obertura basada en un fet real.

## Format de fitxa

```text
Entitat:
Web:
Fet verificat:
Hipotesi comercial:
Angle d'entrada:
Persona/contacte:
Rol:
Risc:
Prioritat recomanada:
Dolor probable:
Pregunta d'obertura:
Fonts:
```

## Regles de rigor

- Una subvencio visible es un fet; el dolor de justificacio es una hipotesi fins que l'entitat ho confirmi.
- Una memoria recent es un fet; que l'equip administratiu vagi saturat es una hipotesi.
- Si el contacte no es operatiu o no esta clar, digues "contacte operatiu no verificat".
- No facis servir llenguatge comercial agressiu ni presentis Summa Social si el briefing nomes es de recerca.
