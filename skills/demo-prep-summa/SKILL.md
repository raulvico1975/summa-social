# Demo Prep Summa

## Principi

No preparis una demo generica. Prepara una conversa amb diagnosi curta i mostra nomes allo que encaixa amb el cas.

## Sortida

```text
Entitat:
Que sabem:
Feina economica probable:
Preguntes de diagnosi:
Funcionalitats a mostrar nomes si encaixen:
Risc comercial:
Seguent pas recomanat:
```

## Preguntes de diagnosi

- Com porteu ara la conciliacio bancaria?
- Teniu socis, donants recurrents o quotes?
- Feu certificats de donacio?
- El model 182 el prepareu internament?
- Les justificacions de subvencions les feu per projecte?
- Treballeu amb Excels separats per projecte, banc i donants?
- Qui acaba revisant la informacio economica abans de presentar-la?

## Funcionalitats a connectar nomes si encaixen

- categoritzacio automatica de transaccions
- gestio de donants
- model 182
- remeses bancaries
- SEPA XML
- justificacio de projectes i subvencions
- multi-divisa per cooperacio internacional
- exportacio i informes

## Proper pas comercial

La sequencia segura es:

1. diagnosi curta
2. demo focalitzada nomes si hi ha encaix
3. implantacio amb dades reals
4. pla mensual
