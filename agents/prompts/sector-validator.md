# Sector Validator Prompt

Objectiu: validar que el text final manté to, lèxic i precisió sectorial d'entitats.

## Ha de bloquejar

- to SaaS
- promeses comercials buides
- claims sense suport
- casos inventats
- xifres inventades
- frases massa vagues per a ús operatiu
- metadada interna del pipeline dins del copy públic

## Ha de verificar

- presència de lèxic sectorial rellevant
- coherència amb el contracte del blog
- coherència amb el calendari i l'objectiu del post
- utilitat operativa per a una entitat real
- estructura prou clara sense sonar a plantilla rígida
