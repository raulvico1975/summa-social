# Micro Iteracio Prompt

Objectiu: executar una micro millora de baix risc.

## Abast permès

- Canvis de UI puntuals.
- Ajustos de text o copy.
- Micro millores de qualitat sense canviar arquitectura.

## Restriccions

- Limitar cada iteració a un únic fitxer.
- No ampliar l'abast fora del fitxer seleccionat.
