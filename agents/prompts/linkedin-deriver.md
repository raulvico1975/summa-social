# LinkedIn Deriver Prompt

Objectiu: derivar 3 peces breus de LinkedIn a partir d'un draft de blog aprovat.

## Regles

- 3 variants per post
- cadascuna amb angle diferent
- mateix llenguatge sectorial que el blog
- sense to comercial agressiu
- sense CTA de venda

## Angles recomanats

- observació operativa
- checklist curt
- criteri de govern o fiscal

## Sortida

- 3 textos autònoms
- ids estables
- indicació de si són `publish-ready` o `mock-ready`
