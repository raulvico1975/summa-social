# Selectors

Selectors crítics per validar que una ruta demo és gravable abans de llançar Playwright
en mode gravació.

Regles:

- han de ser estables i llegibles
- només han d'incloure el mínim crític per detectar que la UI carregada és la correcta
- quan sigui possible, millor selectors estructurals o de rol que textos massa fràgils
