# Scenarios

Cada fitxer d'aquest directori defineix un escenari executable per `video-studio`.

L'escenari ha de descriure:

- ruta inicial
- passos narratius
- estat final esperat
- selector file
- comando de preflight
- comando de gravació
- scene plan reusable

El contracte actual comença per conciliació bancària perquè és el cas més madur i ja
té recorder, storyboard i assets públics alineats.
