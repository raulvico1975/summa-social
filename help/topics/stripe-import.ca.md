## Problema concret
Les donacions cobrades a Stripe no apareixen encara als moviments i cal importar-les correctament.

## Passos exactes
1. Ves a Moviments -> Importar i tria l opcio Stripe.
2. Selecciona el periode que vols importar.
3. Executa la importacio i espera la confirmacio.
4. Revisa els moviments importats i categoritza els pendents.

## Comprovacio final
Les donacions de Stripe apareixen al llistat de moviments amb import i data correctes.

## Error tipic
Importar un periode equivocat i duplicar donacions ja registrades.
