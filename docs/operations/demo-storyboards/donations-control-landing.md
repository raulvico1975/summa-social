# Donations Control Landing

## Objectiu

Peça curta per la landing `control-donacions-ong` i per la capacitat equivalent de la home. Ha d ensenyar que Summa Social centralitza el control de donants i deixa preparat el resum fiscal individual.

## Relat

1. Entrar a `Donants`
2. Mostrar el llistat real de donants
3. Cercar `Clara Soler Ribas`
4. Obrir la fitxa
5. Deixar en pantalla:
   - total històric
   - import net certificable
   - devolució vinculada
   - historial de 3 donacions + 1 devolució
   - accions de certificat

## Cas demo

- Donant: `Clara Soler Ribas`
- Mode seed: `work`
- Dades preparades al seed:
  - 3 donacions fiscals
  - 1 devolució
  - import net certificable: `245,00 €`
