# **App Name**: Summa Social

## Core Features:

- Importación de Extractos: Importación rápida de extractos bancarios en formatos comunes (CSV, OFX).
- Auto-Categorización: Utiliza una herramienta para clasificar automáticamente las transacciones basándose en reglas y patrones predefinidos. Permite al LLM elegir la categoría apropiada según palabras clave, importes y descripciones.
- Asociación de Documentos: Asocia recibos y facturas con transacciones individuales para un mantenimiento de registros organizado.
- Identificación de Documentos Faltantes: Identifica rápidamente las transacciones que carecen de recibos o facturas asociadas.
- Categorías Configurables: Permite a las organizaciones definir categorías personalizadas para ingresos, gastos y transferencias.
- Generación de Informes: Genera informes de resumen mensuales y anuales categorizados por tipo y contacto.
- Exportación de Datos: Exporta datos financieros en formatos adecuados para compartir con auditores u otras partes interesadas (por ejemplo, CSV, PDF).

## Style Guidelines:

- Color primario: Azul celeste suave (#87CEEB) para evocar confianza y fiabilidad.
- Color de fondo: Gris claro (#F0F0F0) para una apariencia limpia y profesional.
- Color de acento: Verde apagado (#90EE90) para resaltar acciones o información importante.
- Fuente del cuerpo y del encabezado: 'PT Sans' (sans-serif) para una sensación moderna pero accesible. Funciona tanto para titulares como para texto del cuerpo.
- Utiliza iconos limpios y simples para representar los tipos de transacciones, las categorías y las funciones de informes.
- Mantén un diseño claro y organizado con un espaciado y una jerarquía visual consistentes para mejorar la usabilidad.
- Utiliza animaciones sutiles (por ejemplo, desvanecimiento, deslizamiento) para proporcionar retroalimentación sobre las interacciones del usuario y guiar al usuario a través de la aplicación.
