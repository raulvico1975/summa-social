# i18n Reports

Aquesta carpeta no conté la configuració funcional principal d'i18n.

La configuració i arquitectura d'i18n és a:

- `docs/i18n.md`
- `src/i18n/*`

## Què hi ha aquí

- `i18n-coverage-es.md` - informe d'auditoria de cobertura d'espanyol
- `es.missing.keys.txt` - claus detectades com a absents a l'auditoria

## Naturalesa dels fitxers

Aquests fitxers s'han d'entendre com a **artefactes d'auditoria o reporting**, no com a contracte funcional.

Es regeneren des de scripts d'i18n quan cal.
