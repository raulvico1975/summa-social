# Product Film Shot Library

This library defines the reusable capture-first shot system for premium product films.

## Rules
- Every shot must be backed by a real capture or a production-approved scene rule.
- Use the shot cards as building blocks, not as final edits.
- Keep copy minimal. If a shot needs explanation to work, reject it and move to a clearer capture.
- Prefer clean motion, stable framing, and a single visual idea per shot.

## Selection rule
Build the film from the fewest shots that can prove the product, show the interface or object clearly, and close with a credible outcome.

## Rejection rule
Reject any shot that relies on invented UI, uncaptured environments, excessive copy, or motion that competes with the product.
